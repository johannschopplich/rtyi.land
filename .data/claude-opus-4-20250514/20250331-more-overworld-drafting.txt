## Game Development Process
- The game is nearing completion with "all 15 main courses are done. But some of them are still awaiting Biomech's rework"
- The overworld is described as "pretty much the last part" that needs to be completed
- The game runs at 60 FPS on real N64 hardware with features like widescreen support and anti-aliasing options
- Beta testing is actively ongoing with testers playing various game modes including "Drunk Mario 64"
- The game is currently around 60MB in size
- Development includes iterative reworking of levels for visual polish

## Game Design Decisions/Challenges
- Major design challenge with the Cloud Cap power-up: "having the cloudcap in the overworld is going to be awkward because what if you just go to a previous part and then you know break the world geometry"
- Implementing overworld events that permanently change the world: "events need to be something you beat and then they give you a star. And then that permanently changes the overworld in some way"
- Addressing coin economy: "I really want to make sure of is that people see value in collecting coins or finding big coin stashes"
- Preventing player overwhelm: Concerned about too many areas unlocking simultaneously after beating the second Bowser level
- Planning final Bowser fight arena that needs to be "flat and empty" for gameplay but not obviously a boss arena beforehand
