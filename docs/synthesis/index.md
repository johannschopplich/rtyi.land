# Documentary Prep

This section contains **curated, cross-stream synthesis** – focused briefings distilled from all 152 stream analyses for documentary work.

::: tip How this works
Run `pnpm stream-synthesis` to generate synthesis files from the per-stream extraction data. The results are written to `.data/synthesis/` and loaded by these pages.
:::

## Sections

| Page                                            | What It Contains                                      | Use Case                            |
| ----------------------------------------------- | ----------------------------------------------------- | ----------------------------------- |
| [**Interview Prep**](/synthesis/interview-prep) | 30–50 curated questions per interviewee, prioritized  | Direct interview preparation        |
| [**Story Highlights**](/synthesis/stories)      | Top 50 narrative arcs ranked by documentary potential | Documentary arc planning            |
| [**Curated Quotes**](/synthesis/quotes)         | Top 80 quotes tagged by use case                      | Narration, trailers, chapter titles |
| [**Topic Narratives**](/synthesis/topics)       | Narrative summary per topic with key findings         | Thematic interview blocks           |
